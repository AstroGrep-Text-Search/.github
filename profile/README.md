**AstroGrep** solves **one specific problem**: locating text patterns inside authorized files when you need line-level context, not just a filename list. Filters, encodings, and result panes keep grepping practical for code, configs, and documents on storage you control.

If you regularly chase strings through projects you maintain, **AstroGrep** is for you. Once a search layout feels familiar, pattern hunts stop feeling like terminal archaeology and become a calm desktop habit.
